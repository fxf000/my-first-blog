	Windows XPでのみ確認

*Font Info
	角川ヘラルド映画「時をかける少女」の作中でタイムリープ中や、
	腕に現れたタイムリープ残数のカウントの数字をフォント化してみました。
	収録文字は数字１〜９、０二種類、：コロンです。
	キーボードの割り当ては下記のキーマップを参考にしてください。
	
	フォント製作にあたって、「時をかける少女」の劇場版予告動画を参考に
	させていただきました。

*Key Map
	数字キー「1」〜「9」はそのまま。
	上位桁の0は「0」
	下位桁の0は「-」に登録されています。
	:は「:」です。
