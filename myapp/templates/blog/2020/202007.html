I wrote a program about Queue to study basic data structures. Queue is a First-In-First-Out (FIFO) data structure that includes operations called enqueue and dequeue. Enqueue is an operation to put data in Queue, and dequeue is an operation to extract data. Then, when the operations of enqueue and dequeue are repeated, the data moves toward the tail, so the time complexity of enqueue and dequeue becomes O(n). To solve this problem, this program used a circular buffer whose time complexity is O(1) as an array to store and retrieve data.

Queue (ALDS1_3_B: Aize Online Judge) 


          import sys

          input = sys.stdin.readline

          class Queue():
              def __init__(self,size):
                  self.head = 0
                  self.tail = 0
                  self.MAX = size+1
                  self.Q = [None for _ in range(self.MAX)]

              def isEmpty(self):
                  return self.head ==  self.tail

              def isFull(self):
                  return self.head == (self.tail+1)%self.MAX

              def enqueue(self,x):
                  if self.isFull():
                      sys.exit("Full")
                  self.Q[self.tail] = x
                  self.tail = (self.tail+1)%self.MAX

              def dequeue(self):
                  if self.isEmpty():
                      sys.exit("Empty")
                  x = self.Q[self.head]
                  self.head = (self.head+1)%self.MAX
                  return x


          def main():
              N,q = map(int,input().split())

              Q_ = Queue(N)
              for n in range(N):
                  p,time = input().split()
                  Q_.enqueue([p,int(time)])

              elapsed_time = 0

              while not Q_.isEmpty():
                  process = Q_.dequeue()
                  if process[1] > q:
                      elapsed_time += q
                      Q_.enqueue([process[0],process[1]-q])
                  else:
                      elapsed_time += process[1]
                      print(process[0]+" "+str(elapsed_time))

          if __name__ == "__main__":
              main()

I wrote a program that implements Doubly Linked List. Doubly Linked List is a data structure that holds a key as a value of a data element and connects the previous and next elements with prev and next pointers. And the each element of the data list is called a Node. The time complexity of adding an element to the Doubly Linked List is O(1), but the time complexity of searching for or deleting a specific element is O(n). However, the deletion of the first or last element is O(1).

Doubly Linked List (ALDS1_3_C: Aize Online Judge) 


          class Node():
              def __init__(self,key):
                  self.key = key
                  self.prev = None
                  self.next = None


          class DoublyLinkedList():
              def __init__(self):
                  self.nil = Node(None)
                  self.nil.prev = self.nil
                  self.nil.next = self.nil

              def insert(self,key):
                  new = Node(key)
                  new.next = self.nil.next
                  self.nil.next.prev = new
                  self.nil.next = new
                  new.prev = self.nil

              def listSearch(self,key):
                  cur = self.nil.next
                  while cur != self.nil and cur.key != key:
                      cur = cur.next
                  return cur

              def deleteNode(self,t):
                  if t == self.nil:
                      return
                  t.prev.next = t.next
                  t.next.prev = t.prev

              def delteFirst(self):
                  self.deleteNode(self.nil.next)

              def deleteLast(self):
                  self.deleteNode(self.nil.prev)

              def deleteKey(self,key):
                  self.deleteNode(self.listSearch(key))


          if __name__ == "__main__":
              from sys import stdin

              input = stdin.readline

              n = int(input())
              DLL = DoublyLinkedList()

              for _ in range(n):
                  com = input().rstrip()
                  if com[0] == "i":
                      key = com[7:]
                      DLL.insert(key)
                  elif com[6] == "F":
                      DLL.delteFirst()
                  elif com[6] == "L":
                      DLL.deleteLast()
                  else:
                      key = com[7:]
                      DLL.deleteKey(key)

              cur = DLL.nil.next

              ans = []
              while cur != DLL.nil:
                  ans.append(cur.key)
                  cur = cur.next

              print(" ".join(ans))

07/28/2020
I wrote two programs about a Binary Search and a Hash Table. The Binary Search is an algorithm that searches a sorted array. And, the binary search looks at the value in the center of the array, compares it to the target value, determines if the target value is to the left or right of the center value, and confirms that it does not exist in one side while exploring. If there are n data, the time complexity for one data of the Binary Search is O (log2n). Then, the Hash Table is a datastructure that can be dynamically inserted, retrieved, and deleted for a set of keyed data. Here, the important point of the Hash table is that the index of the array is determined from the key of the data. The method of determining the index of an array from a key uses a function called a hash function, and obtains the index from the hash value returned from that function. Also, when making the hash function, make sure that different keys do not have the same hash value. This time, in this program, key and m are integers, h1(key) := key mod m and h2(key) := 1 + (key mod (m-1)) are defined, and the hash function h(key,i) is h(key,i) = (h1(key) + i*h2(key)) mod m (i=0,1,···). As the result, the time complexity of inserting and searching is O(1).

Hash Table O(1) O(n)

Binary Search (ALDS1_4_B: Aize Online Judge) 


          def BinarySearch(A,key):
              left = 0
              right = len(A)
              while left < right:
                  mid = (left + right)//2
                  if A[mid] == key:
                      return True
                  elif key < A[mid]:
                      right = mid
                  else:
                      left = mid + 1
              return False

          import sys

          M = 1044697
          H = [None] * M

          table = str.maketrans({"A":"1","C":"2","G":"3","T":"4"})

          def getChar(ch):
              if ch == "A":
                  return 1
              elif ch == "C":
                  return 2
              elif ch == "G":
                  return 3
              elif ch == "T":
                  return 4
              else:
                  return 0

          def getKey(string):
              sum = 0
              p = 1
              for i in range(len(string)):
                  sum += p*(getChar(string[i]))
                  p *= 5
              return sum

          def h1(key):
              return key % M

          def h2(key):
              return 1 + (key % (M-1))

          def find(string):
              i = 0
              key = getKey(string)
              while True:
                  h = (h1(key) + i * h2(key)) % M
                  if H[h] == string:
                      return 1
                  elif H[h] is None:
                      return 0
                  else:
                      i += 1

          def insert(string):
              i = 0
              key = getKey(string)
              while True:
                  h = (h1(key) + i * h2(key)) % M
                  if H[h] is None:
                      H[h] = string
                      break
                  else:
                      i += 1

          input = sys.stdin.readline

          import sys

          n,k = map(int,sys.stdin.readline().split())
          T = [int(sys.stdin.readline()) for _ in range(n)]

          def check(P):
              i = 0
              for j in range(k):
                  s = 0
                  while (s + T[i] <= P):
                      s += T[i]
                      i += 1
                      if i == n:
                          return n
              return i

          left = 0
          right = 100000 * 10000

          while (right - left > 1):
