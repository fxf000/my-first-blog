I wrote two programs. The first program is about Bubble Sort. Bubble Sort is an algorithm which sorts a sequence A in ascending order, and in the worst case, the time complexity is O (n2). If the given sequence A is in order, the best time complexity is O(n). Then, the second program is the one that checks UNIX command. In that program, I made a program to check the number of lines in the file.

Bubble Sort (ALDS1_2_A: Aize Online Judge) 


          def BubbleSort(A,n):
              i = 0
              flag = True
              exchange = 0
              while flag:
                  flag = False
                  for j in range(n - 1, i, -1):
                      if A[j] < A[j-1]:
                          A[j],A[j-1] = A[j-1],A[j]
                          exchange += 1
                          flag = True
                  i += 1
              return A,exchange

          N = int(input())
          S = [int(x) for x in input().split()]

          A,exchange = BubbleSort(S,N)
          print(" ".join(map(str,A)))
          print(exchange)

I wrote three programs. The first program is about Selection Sort for studying algorithms. The worst time complexity is O (n2), which is the same as Bubble Sort. The difference between Bubble Sort and Selection Sort is that Bubble Sort is a Stable Sort that does not exchange when the element keys are the same and maintains the ranking relationship before sorting, whereas Selection Sort is unstable that does not maintain the ranking relationship. The second and third are programs for training UNIX commands, one is a progrma that replaces tabs with spaces, and the other is a program that extracts columns from a text file and saves them.

Selection Sort (ALDS1_2_B: Aize Online Judge) 


          def SelectionSort(A,N):
              count = 0
              for i in range(0,N,1):
                  minj = i
                  for j in range(i+1,N,1):
                      if A[j] < A[minj]:
                          minj = j
                  A[i],A[minj] = A[minj],A[i]
                  if i != minj:
                      count += 1
              return A,count

          N = int(input())
          A = [int(x) for x in input().split()]

          A,count = SelectionSort(A,N)
          print(" ".join(map(str,A)))
          print(count)

I wrote a program about Stable Sort. This program compared Bubble Sort and Selection Sort to see if the ranking relationship was maintained.

Stable Sort (ALDS1_2_C: Aize Online Judge) 


          def BubbleSort(A,N):
              i = 0
              flag = True
              count = 0
              while flag:
                  flag = False
                  for j in range(N-1,i,-1):
                      if A[j][1:] < A[j-1][1:]:
                          A[j],A[j-1] = A[j-1],A[j]
                          count += 1
                          flag = True
                  i += 1
              return A,count

          def SelectionSort(A,N):
              count = 0
              for i in range(0,N,1):
                  minj = i
                  for j in range(i+1,N,1):
                      if A[j][1:] < A[minj][1:]:
                          minj = j
                  A[i],A[minj] = A[minj],A[i]
                  if i != minj:
                      count += 1
              return A,count

          if __name__ == "__main__":
              N = int(input())
              C = list(map(str,input().split()))

              C1 = C
              C2 = C.copy()

              C_B, _ = BubbleSort(C1,N)
              C_S, _ = SelectionSort(C2,N)

              print(" ".join(map(str,C_B)))
              print("Stable")

              print(" ".join(map(str,C_S)))

              if C_S == C_B:
                  print("Stable")
              else:
                  print("Not stable")

Split a file into N pieces (16: NLP100 Exercise) 


          import sys

          def readlines_file(file_name):
              with open(file_name,"r") as f:
                  return f.readlines()

          def writing_file(file_name,text):
              with open(file_name,"w") as f:
                  return f.write(text)

          def suffix(i):
              return "{:03d}".format(i)

          N,file,prefix = sys.argv[1:4]

          lines = list(map(lambda x:x.rstrip(),readlines_file(file)))

          split = len(lines) // int(N)

          for i in range(int(N)):
              offset = i * split
              text = lines[offset : offset + split]
              writing_file(prefix + suffix(i),"\n".join(text))

I wrote a program of Shell Sort. Shell Sort is an algorithm that utilizes Insertion Sort to arrange a list of n elements A in ascending order. Insertion Sort exchanges adjacent elements, while Shell Sort exchanges elements separated by g. Therefore, Shell Sort is not Stable Sort. This time, when I took gi+1 = 3gi+1 (i = 1,2,Â·Â·Â·), the average of time complexity is O (n1.25), but it seems that how to take g and the complexity are still being syudied.

Shell Sort (ALDS1_2_D: Aize Online Judge) 


          def InsertionSort(A,N,g):
              global cnt
              for j in range(g,N,1):
                  v = A[j]
                  k = j - g
                  while k >= 0 and A[k] > v:
                      A[k + g] = A[k]
                      k = k - g
                      cnt = cnt + 1
                  A[k + g] = v

          def ShellSort(A,N):
              h = 1
              while True:
                  if h > N:
                      break
                  G.append(h)
                  h = 3*h + 1
              for i in range(len(G)-1,-1,-1):
                  InsertionSort(A,N,G[i])

          N = int(input())
          A = [int(input()) for n in range(N)]

          G = []
          cnt = 0

          ShellSort(A,N)

          print(len(G))
          print(" ".join(map(str,G[::-1])))
          print(cnt)
          for a in A:
              print(a)

Frequency of a string in the first column in descending order (19: NLP100 Exercise) 


          from itertools import groupby

          file = "./popular-names.txt"

          lines = open(file).readlines()
          lines_list = [line.rstrip().split("\t")[0] for line in lines]
          lines_list.sort()

          result = [(name,len(list(group))) for name,group in groupby(lines_list)]
          result.sort(key=lambda x: x[1],reverse=True)

          for name in result:
              print("name:{name}, count:{count}".format(name=name[0],count=name[1]))

          # UNIX command : cut -f 1 popular-names.txt | sort | uniq -c | sort -r

I wrote a program about Stack. Stack is one of the basic data stractures and, the data is held in Last In First Out structure. Then, the data is moved in and out by two main principle operations, push and pop. In this program, Stack was used to caluculate Four Arithmetic Operations by a method called Reverse Polish Notation. Reverse Polish Notation is a notation where every operator follows all of its operands. For example, a conventional expression (1+2)*(3+4) called Infix notation can be represented as 1 2 + 3 4 + * in the Reverse Polish Notation.

Stack (ALDS1_3_A: Aize Online Judge) 


          def push(x):
              global S
              return S.append(x)

          def pop():
              global S
              return S.pop()

          ope = {"+":lambda a,b: b+a,
                  "-":lambda a,b: b-a,
                  "*":lambda a,b: b*a}

          if __name__ == "__main__":
              S = []

              for s in input().split():
                  if s in ope:
                      push(ope[s](pop(),pop()))
                  else:
                      push(int(s))

              print(" ".join(map(str,S)))
