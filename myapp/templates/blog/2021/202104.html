04/05/2021
I wrote a program to find a diameter of a tree. The tree at this time is a non-negative weighted undirected tree. The diameter of the tree is the maximum distance between the two nodes present in the tree. The algorithm for finding the diameter of the tree first searches for node A, which is furthest from any node. Next look for node B, which is furthest that node A. These nodes A and B are endpoints, and the distance between them is the maximum distance inside the tree. This algorithm is intuitive to be understand, so I will omit the detailed proof. Then time complexity is O(|V|+|E|).

Diameter of a Tree (GRL_5_A: Aize Online Judge) 


          from sys import stdin
          from collections import deque

          input = stdin.readline

          class Edge(object):
              def __init__(self,t,w):
                  self.t = t
                  self.w = w


          def bfs(s):
              global d
              d = [float('inf') for _ in range(n)]
              Q = deque()
              Q.append(s)
              d[s] = 0
              while Q:
                  u = Q.popleft()
                  for i in range(len(G[u])):
                      e = G[u][i]
                      if d[e.t] == float('inf'):
                          d[e.t] = d[u] + e.w
                          Q.append(e.t)

          def solve():
              bfs(0)
              maxv = 0
              target = 0
              for i in range(n):
                  if d[i] == float('inf'):
                      continue
                  if maxv < d[i]:
                      maxv = d[i]
                      target = i
              bfs(target)
              maxv = 0
              for i in range(n):
                  if d[i] == float('inf'):
                      continue
                  maxv = max(maxv,d[i])

              return maxv

          def main():
              global n,G
              n = int(input())
              G = [deque() for _ in range(n)]

              for i in range(n-1):
                  s,t,w = map(int,input().split())
                  G[s].append(Edge(t,w))
                  G[t].append(Edge(s,w))

              ans = solve()

              print(ans)


          if __name__ == "__main__":
              main()
